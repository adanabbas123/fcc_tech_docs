Storing

In programming lanaguges we need to store data in memory. For this purpose we use variables. Variables are like containers for the data.

We use keywords like var, let and const before selecting name for variable.

var a = "a";

Allowed names for javascript variables are:

  • Alphabets
  • Digits (not at the start)
  • Underscore
  • Dollar sign
  • no spaces, no reserved keywords

    • The data we store into variables has two types the primitives(non-modifiable) and the non-primitives(modifiable). Primitive data types are strings, numbers, undefined, null and booleans. Non-pimitives are funtions, objects and arrays etc.

    var string = "string"; let number = 20; var xyz; let bool = true; const abc = null;

    Twisting

    To create a repetitive pattern. We can use three types of loops that are called for loop, while loop and do-while loop.

    For loop helps to create repetition for specific number of times. It is most common loop from other two types.

    for(let i = 0; i < 10; i++){ }

    This loop will end when 10 twists of certain code will be complited.

    On the other hand, while loop continues its loop as long as the code passes the condition of while loop. It never stops. So never use it for your code projects.

    while(condition){ }

    And lastly the do-while always run one time before checking the condition stated inside while portion.

    do{ }while(condition){ }
    Functions

    If we want to separate a certain block of code to perform it again and again at different points. Then functions are best for this kind of role. Functions reduces the lines of code, redundancy. We can call a function anywhere in the code when we need it. It has up to 6 types. Declaration type, anonyms type, expression type, self-invoking type and arrow functions.

    To get the result from a function you must call a function after declaring it.

    function functionName(){ }

    function calling:

    functionName();

    Every type of funtions has new method to call it. But some functions don't need to be called.
